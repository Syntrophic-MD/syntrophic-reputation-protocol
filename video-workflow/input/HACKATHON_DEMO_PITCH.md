# Syntrophic Hackathon Demo Pitch

## One-Line Claim

Syntrophic lets any agent launch as a bonded ERC-8004 identity on Base with one x402 payment.

## Three-Minute Script

### 0:00 - 0:25 Problem

AI agents have a day-zero trust problem.

A new agent might be capable, but it still starts with:
- no portable reputation,
- no economic commitment,
- and too much setup friction.

Today that means onboarding is broken in two ways:
- agents are hard to trust,
- and it takes too many manual steps just to become trusted.

### 0:25 - 0:50 Why Existing Identity Is Not Enough

ERC-8004 gives agents portable identity.

What it does not give them is a fast way to say:

“I am here, I have made a commitment, and any app can verify that onchain.”

That is the gap Syntrophic fills.

### 0:50 - 1:20 Product

Our product is simple:

**Launch a bonded ERC-8004 agent on Base with one x402 payment.**

The user or agent pays once.
Syntrophic then:
- validates the registration payload,
- pins metadata to IPFS,
- registers the ERC-8004 agent,
- posts the ETH bond on Base,
- transfers the identity to the user wallet,
- and returns a proof bundle with the tx hash and bonded status.

So instead of “register now, maybe bond later,” onboarding itself becomes the trust moment.

### 1:20 - 2:15 Demo Flow

Here is the proof flow.

1. We open Syntrophic.MD.
2. We go to Explore Agents.
3. We switch to the Bonded (Demo) filter.
4. We open Syntrophic Agent #222, Frontier Tower.
5. We show:
   - BONDED status,
   - Base registration,
   - ERC-8004 agent ID,
   - and portable syntrophic metadata.

Now this is no longer just a product claim.

It is a bonded ERC-8004 identity with portable trust state already visible onchain.

### 2:15 - 2:40 Why This Matters

This changes behavior immediately.

A newly launched agent can now be:
- discovered as a real ERC-8004 identity,
- verified as bonded,
- and treated differently by tools, marketplaces, and agent ecosystems.

That is valuable because it makes spam cheaper to ignore and credible new agents easier to admit.

### 2:40 - 3:00 Closing

This is not just a dashboard.

It is:
- a live onchain protocol,
- a real agent onboarding workflow,
- a real payment flow,
- and a simple entry point into the ERC-8004 ecosystem.

But the invitation starts now:

**Is your OpenClaw on 8004 yet? Send it to Syntrophic.MD and it will register itself, stake its reputation, and become a trusted agent ready for the coming agent economy.**
