## Workflow Steps

### Step 0: Generate Audio (`00-generate-audio.js`)

**What it does:**
1. Connects to ElevenLabs API
2. Generates voiceover with Frank (friendly deep male voice)
3. Use upbeat Apple/Google ad-style background music
4. Synthesizes all sound effects (click, whoosh, chime)

**Dependencies:**
- ElevenLabs API key (stored in macOS keychain)
- FFmpeg (for SFX synthesis)

**Outputs:**
- `assets/voiceover.mp3` - Main narration
- `assets/background-music.mp3` - Background music
- `assets/sfx/*.mp3` - All sound effects

---

### Step 1: Generate Slides (`01-generate-slides.js`)

**What it does:**
1. Creates HTML slides from `config.json`
2. Takes screenshots with Playwright
3. Converts to MP4 videos with proper duration
4. Adds fade animation

**Slides:**
- **Opening** (3s): Title + subtitle with fade-in
- **Closing** (8s): Title + tagline + CTA with fade-out

---

### Step 2: Record Demo (`02-record-demo.js`)

**What it does:**
1. Launches headless Chromium
2. Navigates to `https://syntrophic.md`
3. Clicks "Open App" button
4. Waits for new page (handles both popup and same-page nav)
5. Waits 5 seconds on app page
6. Records video with Playwright
7. Trims to target duration (19s)

**Configurable Actions:**
```json
"demo.actions": [
  { "type": "navigate", "url": "...", "wait_after": 2000 },
  { "type": "click", "selector": "text=Open App", "new_page_wait": 5000 },
  { "type": "scroll", "direction": "down", "amount": 400 }
]
```

---

### Step 3: Compose Video (`03-compose-video.js`)

**What it does:**
1. Converts WebM demo to MP4
2. Trims demo to target duration
3. Concatenates: opening + demo + closing
4. Mixes 7 audio tracks:
   - Background music (volume: 0.2)
   - Voiceover (start: 0s)
   - Click SFX (on button click)
   - Whoosh SFX (on transitions)
   - Chime SFX (on closing start)
5. Applies audio fades
6. Outputs final MP4

Edit `config.json`:
```json
"audio.music.url": "https://your-music-url.mp3"
```

Or replace `assets/music.mp3` directly.

## Troubleshooting

### "Open App" doesn't work
The script handles both popup and same-page navigation. Check that the button text matches.

### Audio too quiet/loud
Adjust `volume` values (0-1 scale):
```json
"audio.music.volume": 0.15,      // Lower background
"audio.voiceover.volume": 1.0,   // Max voice
```

### Video duration wrong
Check `timing` section totals match `total_target`.

### ElevenLabs API fails
Verify API key:
```bash
security find-generic-password -a 'friendlyagent222@gmail.com' -s '11 Labs API Key' -w
```

### No sound effects
Re-run audio generation:
```bash
node scripts/00-generate-audio.js
```
